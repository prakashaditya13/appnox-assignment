# Appnox Search Screen Assignment

This Repository contains only Apponox Search Screens design codes.

## Build Using

This Project is built using React + Vite

## Installation

1) Clone the repo.
2) ```
   npm install
   ```
3)  ```
    npm run dev
    ```
